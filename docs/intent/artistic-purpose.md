# Artistic Purpose

mazeASK treats maze generation as a medium for computational form.

Its artistic intent lies in:

- visual signature produced by small rule changes
- constrained variation across shared structure
- structurally honest abstraction rather than decorative novelty
- computational form that remains legible as a system

Maze generation is treated here as a form-making practice.

The work studies how different selection rules, within a constrained colorsASK palette and a modularized visual system, produce distinct families of structure, rhythm, density, and flow.

Color introduces constrained randomness inside a bounded palette so that each run can shift in tone while remaining recognizably part of the same visual language.

In this sense, mazeASK follows an ASK visual logic >> order from chaos // beauty in systems

mazeASK therefore treats visual style as part of the work's meaning:

- modular enough to vary
- constrained enough to remain coherent
- abstract enough to compare
- honest enough to remain tied to structure
